# دليل بناء أداة Gps Wolfox

هذا المجلد يحتوي على الكود المصدري وإعدادات بناء الأداة. لإنتاج ملف `.deb` الجاهز للتثبيت، اتبع الخطوات التالية:

## المتطلبات
يجب أن يكون لديك أداة **Theos** مثبتة على جهازك (سواء كان ماك، لينكس، أو آيفون بجيلبريك).

## خطوات البناء (عبر Terminal)
1. افتح تطبيق Terminal وانتقل إلى مجلد المشروع:
   ```bash
   cd path/to/GpsWolfox
   ```
2. قم بتشغيل أمر البناء:
   ```bash
   make package
   ```
3. بعد الانتهاء، ستجد ملفاً جديداً بصيغة `.deb` داخل مجلد يسمى `packages`.

## كيفية التثبيت
بعد الحصول على ملف الـ `.deb`:
1. انقل الملف إلى هاتفك.
2. افتحه باستخدام تطبيق **Filza**.
3. اضغط على **Install** ثم قم بعمل **Respring**.

---
**ملاحظة:** تم إعداد هذا المشروع ليدعم معالجات A12 فما فوق (arm64e) والمعالجات الأقدم (arm64).
